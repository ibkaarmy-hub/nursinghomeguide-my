# WhatsApp — first-touch outreach to operator

Use this for the initial WhatsApp message to a facility (per verification-sop.md §3 step 3). Preferred channel — WhatsApp response rates run higher than email for Malaysian SME operators.

Placeholders:
- `<facility-name>` — facility name as it appears on the public profile
- `<facility-url>` — the canonical `/facility/<slug>/` URL
- `<form-url>` — pre-filled Google Form link for this facility
- `<helper-name>` — your name + role (e.g. "Aisyah, verification team")

---

`<facility-url>`

`<form-url>`

## Operator notes (helper-only — do not send)

- Don't promise a timeline beyond "within a few working days"
- Don't claim we'll publish their licence number — we don't
- If they ask who runs the site: "NursingHomeGuide.my is run by a team of Malaysian doctors and researchers. I'm part of the verification team."
- Never name any individual as running the site — sign with helper name only
