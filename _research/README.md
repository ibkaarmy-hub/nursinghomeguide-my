## Conventions

- Every claim cites a primary source (URL).
- Date the file (`Last updated: YYYY-MM-DD`) so we know when to refresh.
- If something can't be verified, say so explicitly. Don't paper over gaps.
- Don't quote >15 words from any source — paraphrase + link.
